# kolam-design-classification-cnn-
CNN-based Kolam image classification system integrated with a responsive web interface and AI chat assistant.
Kolam AI is a full-stack AI-powered web application that identifies traditional Kolam designs using a Convolutional Neural Network (CNN) and provides an interactive chat interface for learning about Kolam art and cultural significance.

Project Overview

This project combines Computer Vision, Deep Learning, and Web Development to digitally preserve and analyze traditional Kolam patterns.

Users can upload Kolam images, get automatic pattern classification, and explore cultural insights through an interactive chat assistant.

The web interface is designed with responsive UI, smooth animations, and modern styling to enhance user experience.
